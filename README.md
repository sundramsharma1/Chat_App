### Run Locally

To run **Chat App** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/sundramsharma1/Chat_App.git
```

Windows:

```bash
git clone https://github.com/sundramsharma1/Chat_App.git
```
